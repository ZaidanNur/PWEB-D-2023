HTML

Hyper Text Markup Language

HTML terdiri dari beberapa bagian yang biasanya disebut dengan HTML element yang diawali dengan tag pembuka diakhiri dengan tag penutup

Berikut merupakan element root yang bisa kita buat untuk menginisiasi dokumen HTML

<!DOCTYPE html> [Mark untuk dokumen HTML5]
<html> [Root element dari html]
<head> [Berisikan informasi terkait dokumen html]
<title>Title</title> [Judul dari dokumen html]
</head>
<body> [Bagian isi dari dokumen yang nanti akan ditampilkan]
</body>
</html>

Form dan Input

Masukan dan submission

<form action="index.html">
<label for="nama">Nama</label>
<input type="text" id="nama">
<label for="kelas">Kelas</label>
<input type="text" id="kelas">
<button type="submit">Submit</button>
<label for="alasan">Alasan Pengunduran diri</label>
<textarea name="" id="alasan" cols="30" rows="5"></textarea>
</form>

action = tujuan pengiriman data

<label> = label / nama untuk inputan

for = tujuan input (id dari input)

<input> = area untuk submission

type = tipe masukan dari input

id = id dari inputan

<textarea> = area masukan yang lebih ke deskripsi

name = keterangan inputan

cols dan row = luas dari area masukan

<button> = tombol

type = fungsi tombol

Style <style>

Menggunakan css pada html

Letaknya ada pada head, bukan body

*contoh ada di bagian Selector

Selector

Selector berfungsi untuk filter penggunaan css pada element tertentu

<head>
<style>

Universal Selector
Selector yang mempengaruhi seluruh dokumen (tanda *)

*{ margin:0px }

Element Selector
Selector yang mempengaruhi pada setiap element yang sama (menggunakan element)

p{ color:red; }

Class Selector
Selector yang mempengaruhi pada setiap element dengan class tertentu (tanda . diikuti nama class)

.nama_class{ padding: 0px }

ID Selector
Selector yang mempengaruhi pada element dengan id tertentu (tanda # diikuti id)

#id { color:white }
</style>
</head>

Style juga bisa diterapkan langsung tanpa selector, yaitu dengan menyisipkan pada element tertentu (Inline Style)

<p style="color:white"> Ini paragraf </p>

Untuk selector ini ada hirarki atau urutannya berdasarkan spesifiknya

  1. Inline Style i.e <p style="color:white">
  2. ID Selector i.e #id
  3. Class Selector, Attribute Selector, dan Pseudo-class Selector i.e .class [attribute] :pseudo-class
  4. Element Selector dan Pseudo-element Selector i.e h1 ::after
